# Java Exception Handling Demonstration

This Java console application is designed to help beginners understand how to handle common exceptions in Java using `try-catch` blocks. It provides a simple menu to test different types of exceptions and shows how to handle them gracefully.

## Features

- Arithmetic Exception (Divide by zero)
- Array Index Out Of Bounds
- Number Format Exception
- Null Pointer Exception
- Input Mismatch Exception
- Clean program exit

## How to Use

1. Clone or download this repository.
2. Open the project in your favorite IDE (like Eclipse or IntelliJ) or use the command line.
3. Compile the Java file:
   ```bash
   javac Main.java
